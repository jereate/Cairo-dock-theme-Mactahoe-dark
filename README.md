# Cairo-Dock MacTahoe-Dark Custom Theme

Este es un tema personalizado para Cairo-Dock basado en el estilo **MacTahoe-Dark**. Ofrece una estética oscura, elegante y moderna inspirada en macOS.

## 🚀 Requisitos Previos

Antes de instalar, asegúrate de tener instalado Cairo-Dock y sus complementos:

```bash
sudo apt install cairo-dock cairo-dock-plug-ins
```

## 📥 Instalación Paso a Paso

### Método 1: Desde la interfaz de Cairo-Dock (Recomendado)
1. Descarga el archivo comprimido de este repositorio (`.tar.gz` o `.zip`).
2. Haz **clic derecho** sobre tu Cairo-Dock actual.
3. Ve a **Cairo-Dock** > **Configurar** (Configure).
4. Selecciona la pestaña **Temas** (Themes) en la parte superior.
5. Haz clic en el botón **Abrir / Importar** (icono de carpeta).
6. Selecciona el archivo comprimido que descargaste.
7. Haz clic en **Aplicar** (Apply).

### Método 2: Instalación Manual (Alternativa)
Si prefieres instalarlo manualmente moviendo los archivos:
1. Extrae el contenido del archivo comprimido.
2. Mueve la carpeta del tema a la ruta de temas locales de Cairo-Dock en tu directorio personal:
   ```bash
   ~/.config/cairo-dock/themes/
   ```
3. Reinicia Cairo-Dock, ve a la configuración de **Temas** y selecciónalo en la lista de temas locales.

## 🛠️ Notas de Personalización
* **Iconos:** Si usas un paquete de iconos específico para complementar este tema, se recomienda activarlo globalmente en tu sistema operativo.
* **Efectos:** Para obtener el efecto de reflejo exacto de las capturas de pantalla, asegúrate de tener activada la aceleración por hardware (OpenGL) en la configuración de Cairo-Dock.
